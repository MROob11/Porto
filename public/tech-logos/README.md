## Format Rekomendasi

- **SVG** (lebih baik untuk scaling)
- atau **PNG** dengan background transparent
- Ukuran: 64x64px atau 128x128px minimum

## Cara Download Logos

Kamu bisa download dari:

1. **SVG Repo**: https://www.svgrepo.com/
2. **Simple Icons**: https://simpleicons.org/
3. **Official websites** masing-masing tech
4. **Worldvectorlogo**: https://worldvectorlogo.com/

## Notes

- Pastikan semua nama file **lowercase**
- Format nama: `namatech.svg` atau `namatech.png`
- No spaces in filename!
